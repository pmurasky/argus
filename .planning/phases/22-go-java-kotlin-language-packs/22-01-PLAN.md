         identifiers, lowercase for unexported; no `utils`/`helpers`/`common` grab-bag packages;
         use the `slices` and `maps` stdlib packages (Go 1.21+) over manual index arithmetic;
         name package contents to read well at the call site (avoid stutter like `http.HTTPServer`).
